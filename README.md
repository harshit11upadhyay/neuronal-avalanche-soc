# Neuronal Avalanches and Self-Organized Criticality (SOC)

This project simulates neuronal avalanche dynamics using a threshold-based cascade model on complex networks, demonstrating self-organized criticality (SOC).

Overview
Neuronal avalanches are bursts of activity in the brain that follow power-law distributions. This project models such dynamics using a network-based system.


Model Features
- Threshold-based firing
- External noise (slow driving)
- Avalanche cascade mechanism
- Network connectivity (random & scale-free)


Key Results
- Power-law distribution: α ≈ 1.5
- Scaling relation: ⟨s⟩ ~ T^γ
- SOC behavior confirmed


How to Run
pip install numpy matplotlib networkx scipy  
python code/complexity_brain_avalanche.py
